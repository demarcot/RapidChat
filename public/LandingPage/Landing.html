About Rapid Chat

Rapid Chat is a chat application that can be hosted locally or remotely and is supports the following types of communication: public chat rooms, private chat rooms, and direct messages.

Connect Anywhere!

Rapid Chat has been optimized use with mobile devices and desktops.

No Limits!

Rapid Chat's database does not inherently have any limitations on the number of users, the number of chat rooms, or the number of messages. If the hardware can hold the data, Rapid Chat will handle the rest!

Local and Remote
servers

Rapid Chat servers can be hosted on your own local device! See the README file on GitHub for more details. Remote servers can be created by contacting [email protected]

Rapid Chat Box

The Rapid Chat Box is a Raspberry Pi that comes pre-installed with all the necessary software to host the Rapid Chat server and database.
