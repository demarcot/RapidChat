Rapid Chat

Let the people speak

Rapid Chat is a chat application. The application is available on all platforms. Rapid Chat servers can be hosted remotely or locally. There is no limitation on the number of users, the number of messages sent or received, or the size of the database. Rapid chat is free of charge to all users. The application does not include a file transfer system or voice communication.
